# SteamJavascriptHelpers
This is helper javascript scripts that can be injected into store.steampowerd.com with an extension called <a href="https://violentmonkey.github.io/">"Violentmonkey"</a>.

To install follow <a href="https://violentmonkey.github.io/guide/creating-a-userscript/">this guide</a> then copy and paste one of these scripts.
